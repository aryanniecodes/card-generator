# Card Generator (DOM Project)

This is a simple JavaScript DOM project where users can generate a profile card by entering details like image URL, name, occupation, and info.
For Educational Purpose

## Features

* Dynamic card creation using JavaScript
* Form validation
* Responsive UI
* Image fallback handling

## Tech Used

* HTML
* CSS
* JavaScript (DOM)

## Purpose

This project is built for learning and practicing DOM manipulation and basic frontend development.
